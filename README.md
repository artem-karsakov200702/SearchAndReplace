# Text Search & Replace

**Text Search & Replace** — консольное приложение на C++ для генерации, поиска и замены текста в JSON-файлах.

Программа позволяет:
- Генерировать JSON-файлы (корректные и с ошибками) `data_0.json...data_N.json`
- Искать текст с поддержкой case-sensitive/insensitive режимов
- Выполнять массовую замену текста во всех файлах
- Выводить таблицы результатов и замеры производительности
- Обрабатывать 100 000+ файлов (~70GB данных)

## Запуск программы

Скачай репозиторий и перейди в папку `release/`.  
Дважды кликни на `SearchAndReplace.exe`.

После запуска появится меню:

1.) Генерация 2) Поиск/замена 3) Выход [help/h]

Генерация JSON - создаёт data_0, data_1... (~0.7MB каждый)

Поиск/замена - ищет/заменяет текст во всех объектах

Выход - завершить программу

help / h - показать справку
