## 1. Краткое описание задачи

Разработать консольное приложение на языке C++, предназначенное для поиска и замены текста в JSON-файлах.

Программа выполняет:
- Чтение JSON-файлов с объектами {name, content}
- Генерацию тестовых JSON-файлов (с ошибками и без ошибок) 
- Поиск текста с поддержкой case-sensitive/insensitive режимов (учёт регистра)
- Массовую замену текста во всех файлах
- Сохранение результатов обратно в JSON
- Измерение производительности обработки

## 2. Цель и ожидаемый результат

**Цель работы** — освоить работу с JSON в C++, эффективные алгоритмы поиска/замены строк, массовую обработку файлов и бенчмаркинг производительности.

**Ожидаемый результат:**
Консольное приложение (.exe), которое:
- Генерирует тысячи JSON-файлов (~0.7MB каждый)
- Обрабатывает 100 000+ файлов за ~5 минут
- Поддерживает режимы поиска и замены
- Выводит таблицы результатов и замеры времени
- Корректно обрабатывает некорректные JSON

## 3. Этапы и задачи реализации

**Анализ формата входных данных**
- Определение структуры JSON: массив объектов с полями "name", "content"
- Выделение обязательных полей и допустимых типов данных

**Проектирование структуры программы**
Разделение кода на модули:

SearchAndReplace/   
├── /src/ # исходный код .cpp    
├── /include/ # заголовочные файлы .h/.hpp    
├── /tests # самотесты    
├── /docs/ # отчёт, план реализации, бенчмарки    
├── /data/ # примеры JSON    
├── /release/ # готовый .exe    

**Реализация генерации файлов**
- Генерация корректных JSON-файлов data_0.json...
- Создание файлов с искусственными ошибками (незакрытые массивы, пустые объекты)
- Генерация текста с ключевыми словами "foo", "bar", "test"

**Парсинг и обработка данных**
- Использование библиотеки [nlohmann/json](https://github.com/nlohmann/json)
- Извлечение объектов TextObject {name, content}
- Пропуск некорректных объектов с логированием

**Алгоритмы поиска и замены**
- findAllPositions(): алгоритм string::find() с case-insensitive
- replaceAllOccurrences(): сбор результата по позициям с reserve()
- Массовое сканирование файлов data_0...data_99999.json

**UX и консольный интерфейс**
- Интуитивное меню: 1) Генерация 2) Поиск/замена 3) Выход
- Подробная справка через help/h
- Таблицы результатов для <50 объектов
- Замеры времени обработки

**Сохранение результатов**
- Восстановление оригинальной JSON-структуры
- Обновление только полей content
- Сохранение в исходные файлы

**Тестирование производительности**
- Бенчмарки для 100, 1000, 10000, 100000 файлов
- Сравнение режимов поиска/замены
- Анализ линейной зависимости времени
